## 加载资产包

开发环境

```
https://demo2.steedos.cn?assetUrls=http://127.0.0.1:8080/@steedos-widgets/amis-object/dist/assets-dev.json,http://127.0.0.1:8080/@steedos-widgets/devextreme/dist/assets-dev.json

```

